# ImageProjet
Based on: Real-Time and Accurate Drone Detection in a Video  with a Static Background

From:
[1] Seidaliyeva, Ulzhalgas & Akhmetov, Daryn & Ilipbayeva, Lyazzat & Matson, Eric. (2020). Real-Time and Accurate Drone Detection in a Video with a Static Background. Sensors. 20. 3856. 10.3390/s20143856. 


Training  set:

[2] F. Svanström, «DroneDetectionThesis/Drone-detection-dataset: First release». Zenodo, may 16, 2020. doi: 10.5281/zenodo.5500576.


Testing Data Set:

[3] Sun, Z.-W., Hua, Z.-X., Li, H.-C., Qi, Z.-P., Li, X., Li, Y., & Zhang, J.-C. (2025). FBD-SV-2024: Flying Bird Object Detection Dataset in Surveillance Video. Scientific Data, 12(1), 530. https://doi.org/10.1038/s41597-025-04872-6
